네이버에서 주식 정보 추출하기
=============================

종목코드를 넣어주면 해당 종목에 대한 현재 가격과 거래량을 반환한다.

엑셀과 같은 Spreadsheet 와 함께 사용하면 종목 분석에 활용할 수 있으리라 기대한다.

---------
